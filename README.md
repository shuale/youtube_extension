### To use this extension:

1. Download the `.json`, `.js` and `.png` files and place them into new folder.
2. Go to the *extensions* page in your Chrome (chrome://extensions) or Edge (edge://extensions) browser.
3. Turn on **Developer mode** (toggle is in the top-right corner in Chrome and top-left corner in Edge).
4. Click the **Load unpacked** button near the top of the page.
5. In the popup window, select the folder you created in **Step 1**.
